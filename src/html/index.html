Assignment Due Date (Backend)

On this page, you will find a short description of available endpoints.

GET Requests

  • To retrieve all modules - /api/modules

    Returns an array of objects that will contain all modules

    Module properties:"_id" , "name", "image_link", "code"

  • To retrieve all assignments - /api/tasks

    Returns an array of objects with assignments

    Task properties:"_id" , "name", "setdate", "duedate", "finaldeadline", "modulecode"

  • To retrieve all assignments filltered by module - /api/tasks/{modulecode}

    Returns an array of objects with assignments

    Task properties:"_id" , "name", "setdate", "duedate", "finaldeadline", "modulecode"
